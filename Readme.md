# CATLOVER-API

> API서버.

크롤링할 대상과 프론트에서 보여줄것을 전송해주는 api 서버<br>
Express, NodeJS, Mongoose, PUG TemplateEngine 사용

-   CATLOVER 페이지 - [[링크]](http://www.catloverlife.com/)
